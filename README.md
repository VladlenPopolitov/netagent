# netagent

**netagent** is a lightweight network management daemon for FreeBSD systems, designed for desktop use.

It provides a simple and secure interface (via a Unix socket) for configuring network interfaces such as Ethernet and Wi-Fi. A small CLI and Python library are included for interacting with the daemon.

The project aims to offer a minimal, clean alternative to complex network managers, with a focus on security, simplicity, and extensibility.

⚠️ Work in progress — interfaces and features may change.
